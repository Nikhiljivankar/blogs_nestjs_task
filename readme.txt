requirements: 
nodejs installed in system
mongodb compass  installed
npm install in system



Install all the dependencies using below command:

npm install 

create .env file  in root directory and add following content to it. Replace <your-mongodb-uri> with your own MongoDB URI. port review  .env.example file

To run the application, use this command in your terminal or cmd :

npm run start

to see swagger documentation use following link in browser:
http://localhost:3000/api/#


Description:
It is a blog on Nest.js(Passport(JWT+Local)) and MongoDB(Mongoose). You can:
1. Login, Register
2. Create, read, update and delete posts.
3. Create, read comments.
4. Read user profiles.


please use swagger file to  understand how API's are working.
